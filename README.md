# Projektas Tinklapių pritaikomumas

Mokomasis projektas, sukurtas naudojant HTML.
Nuoroda į projektą: [Projektas "Tinklapių pritaikomumas"] (https://github.com/AminaJudenaite/7-projektas)

## Naudotos technologijos
* HTML

## Kaip paleisti projektą
1. Atsisiųskite projekto failus.
2. Atidarykite `index.html` failą savo interneto naršyklėje.

## Testavimas
* Klaviatūros testas: Ar galite pasiekti visus elementus spaudžiant tik Tab? Ar
matote ryškų fokusą (focus ring)? TAIP
* NVDA/VoiceOver testas:
• Ar ekrano skaitytuvas praneša, kad „Ieškoti“ yra mygtukas? TAIP
• Ar pranešama, ar išskleidžiamas meniu yra „išskleistas“, ar „suskleistas“? TAIP
